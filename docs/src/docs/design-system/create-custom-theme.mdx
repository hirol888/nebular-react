# Create Custom Theme

## Select parent Theme

In order to minify the amount of work, parent theme selection is a very important step.
Though it is very simple - for colored and dark themes - use `DARK_THEME` base theme, otherwise - `DEFAULT_THEME` theme.
In our case `MY_THEME` theme is a colored one, let's use `DARK_THEME` as a parent.

<hr />

## Create a custom theme

To create a custom theme, we need to create a `MyTheme.ts` in your `src` folder.

```ts
import { DEFAULT_THEME, NebularTheme } from '@nebular-react/core';
import { Property } from 'csstype';

export interface MyTheme extends NebularTheme {
  my_padding: Property.Padding;
  ...
}

export const MY_THEME: MyTheme = {
  ...DEFAULT_THEME,
  my_padding: '1.25rem 0',
  // new primary color
  color_primary_100: '#faf7ff',
  ...
};
```

To use the created theme, in your `App(.tsx/.ts/.jsx/.js)`, wrap up with `NebularProvider`.

```ts
import { NebularProvider } from '@nebular-react/styles';
import { MY_THEME } from './MyTheme';

export default function App() {
  return <NebularProvider theme={MY_THEME}>...</NebularProvider>;
}
```

Now the new theme is enabled, let's modify it.

<hr />

## Start with Basic

Basic shades are one of the most important as they are the source for backgrounds, borders and text colors.
In the dark theme, text colors utilize the upper part of the shades (whitish colors)
and backgrounds the lower part.

Let's modify them and put our colors, starting with white for `color_basic_100` and finishing with almost black
for `color_basic_1100`:

```ts
import { DARK_THEME, NebularTheme } from '@nebular-react/core';
import { Property } from 'csstype';

export interface MyTheme extends NebularTheme {
  my_padding: Property.Padding;
  ...
}

export const MY_THEME: MyTheme = {
  ...DARK_THEME,
  my_padding: '1.25rem 0',
  // new basic color
  color_basic_100: 'white',
  color_basic_200: '#eefafc',
  color_basic_300: '#e8f5fa',
  color_basic_400: '#d6e6f2',
  color_basic_500: '#c7dbeb',
  color_basic_600: '#a9c0db',
  olor_basic_700: '#4f969e',
  color_basic_800: '#336a77',
  color_basic_900: '#295c66',
  color_basic_1000: '#244555',
  color_basic_1100: '#121a2b'
  ...
};
```

<hr />

## Tweak backgrounds

In a case we need to tweak the color more accurately, we can change how basic colors are used.
For example, we can make the theme backgrounds and borders lighter, by using basic color lighter for one grade up:

```ts
import { DARK_THEME, NebularTheme } from '@nebular-react/core';
import { Property } from 'csstype';

export interface MyTheme extends NebularTheme {
  my_padding: Property.Padding;
  ...
}

export const MY_THEME: MyTheme = {
  ...DARK_THEME,
  my_padding: '1.25rem 0',
  // new basic color
  color_basic_100: 'white',
  color_basic_200: '#eefafc',
  color_basic_300: '#e8f5fa',
  color_basic_400: '#d6e6f2',
  color_basic_500: '#c7dbeb',
  color_basic_600: '#a9c0db',
  color_basic_700: '#4f969e',
  color_basic_800: '#336a77',
  color_basic_900: '#295c66',
  color_basic_1000: '#244555',
  color_basic_1100: '#121a2b',
  background_basic_color_1: 'var(--color-basic-700)',
  background_basic_color_2: 'var(--color-basic-800)',
  background_basic_color_3: 'var(--color-basic-900)',
  background_basic_color_4: 'var(--color-basic-100)',
  border_basic_color_1: 'var(--color-basic-700)',
  border_basic_color_2: 'var(--color-basic-800)',
  border_basic_color_3: 'var(--color-basic-900)',
  border_basic_color_4: 'var(--color-basic-1000)',
  border_basic_color_5: 'var(--color-basic-1100)'
  ...
};
```
