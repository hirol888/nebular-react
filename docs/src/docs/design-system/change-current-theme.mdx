# Change Current Theme

Nebular Theme System provides 4 color schemes out of the box - `DEFAULT_THEME`, `DARK_THEME`, `CORPORATE_THEME` and `COSMIC_THEME`.

## Runtime Theme Switch

In case you want to have better control when a theme is changed, or for instance need to change it based on a user role,
it is possible to dynamically tell Nebular which theme should be enabled.

```ts
import { changeTheme, COSMIC_THEME } from '@nebular-react/core';

export function Component() {
  return <button onClick={() => changeTheme(COSMIC_THEME)}>Change To Cosmis Theme</button>;
}
```
