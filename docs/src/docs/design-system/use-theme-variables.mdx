# Use Theme Variable

In case when you need to access theme variables from your application component,
Nebular provides you with simple sass function called `createStyles`.
It returns an `CSSObject` that can be used in `(.tsx/.ts/.jsx/.js)`.

<hr />

## Create styles for a component

Create `MyComponent.styles.ts` in your component folder.

```ts
import { createStyles } from '@nebular-react/core';
import { MyTheme } from '../MyTheme';

export default createStyles<MyTheme>((theme) => ({
  root: {
    position: 'relative',
    padding: theme.my_padding,
    ...
  }
}));
```

Then in `MyComponent.tsx`, use the styles you created.

```ts
import useStyles from './MyComponent.styles';

export function MyComponent(props: MyComponentProps) {
  const { classes, cx } = useStyles();

  return <div className={cx(classes.root)}></div>;
}
```
