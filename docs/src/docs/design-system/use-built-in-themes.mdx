## Use a built-in theme

In your `App(.tsx/.ts/.jsx/.js)`, wrap up with `NebularProvider`. By default, `DEFAULT_THEME(light)` is used, you can choose other predefined themes, `CORPORATE_THEME`, `COSMIC_THEME`, `DARK_THEME`

```ts
// default(light) theme
import { NebularProvider } from '@nebular-react/styles';

export default function App() {
  return <NebularProvider>...</NebularProvider>;
}
```

```ts
// corporate theme
import { NebularProvider, COSMIC_THEME } from '@nebular-react/styles';

export default function App() {
  return <NebularProvider theme={COSMIC_THEME}>...</NebularProvider>;
}
```

```ts
// cosmic theme
import { NebularProvider, CORPORATE_THEME } from '@nebular-react/styles';

export default function App() {
  return <NebularProvider theme={CORPORATE_THEME}>...</NebularProvider>;
}
```
