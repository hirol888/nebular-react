# What is Nebular React?

Nebular React is a customizable React Library based on Eva Design System specifications, with 40+ UI components and 4 visual themes. It is TypeScript based and all components and hooks export types.

<hr />

## What's included

Nebular React modules are distributed as separated `npm` packages:

- Nebular React Core `@nebular-react/core`
  - Core components library, exports everything from `@nebular-react/styles`
- Nebular React Hooks `@nebular-react/hooks`
  - Collection of hooks for state and UI management
- Nebular React Styles `@nebular-react/styles`
  - [Emotion](https://emotion.sh/) based css-in-js library that is used in all components
  - 4 visual themes, a customizable theming engine
- Nebular React Next `@nebular-react/next`
  - Server side rendering utilities for Next.js
- Nebular React SSR `@nebular-react-ssr`
  - Server side rendering utilities

<hr />

## Browser support

Nebular React supports most recent browsers.

## Assumptions

This documentation assumes that you are already familiar with JavaScript/TypeScript, React, CSS-IN-JS.
