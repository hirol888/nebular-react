# Register Icon pack

Nebular comes with an `<Icon></Icon>` component that accepts an `icon="icon-name"` and `pack="icon-pack"` parameters.

By default, Nebular includes only a small and limited pack called `nebular-essentials`. This pack consists of essential icons, such as `close-outline`, `checkmark-outline`, etc that are required by the Nebular components (modals, accordions, etc).

## Eva Icons

In case you need to have more icons available, Nebular provides a `eva-icons` pack with 480+ SVG icons.

Install the pack:

```bash
# With yarn
yarn add eva-icons

# With npm
npm i eva-icons
```

This command will install Eva Icons pack. The default icon pack name is `eva`, which means you can render a Eva icon without specifying the pack implicitly:

```html
<Icon icon="star" />
```

## 3rd-party/Custom SVG Pack

In case you need to use any other 3rd party SVG Icon pack, here is the example of how to register ionicons:

First, add these css

```css
.ionicon {
  fill: currentColor;
  stroke: currentColor;
}
.ionicon-fill-none {
  fill: none;
}
.ionicon-stroke-width {
  stroke-width: 32px;
}
```

Then extend defaultSvgIconPacks:

### Outside of NebularProvider

```ts
import { defaultSvgIconPacks, SvgFile, NebularProvider } from '@nebular-react/styles';
import * as ionicons from 'ionicons/icons';

export function App() {
  const _ionicons = Object.entries<string>(ionicons).map(
    (x: [string, string]): [string, SvgFile] => {
      const name = x[0].replace(/[A-Z]/g, (m) => `-${m.toLowerCase()}`);
      return [name, { file: `https://unpkg.com/ionicons@7.1.0/dist/ionicons/svg/${name}.svg` }];
    }
  );

  const svgIconPacks = new Map(defaultSvgIconPacks);
  svgIconPacks.set('ion', new Map(_ionicons));

  return <NebularProvider svgIconPacks={svgIconPacks}>...</NebularProvider>;
}
```

### In children components of NebularProvider

```ts
import { useNebularIconPacks } from '@nebular-react/styles';
import * as ionicons from 'ionicons/icons';

export function SomeComponent() {
  const _ionicons = Object.entries<string>(ionicons).map(
    (x: [string, string]): [string, SvgFile] => {
      const name = x[0].replace(/[A-Z]/g, (m) => `-${m.toLowerCase()}`);
      return [name, { file: `https://unpkg.com/ionicons@7.1.0/dist/ionicons/svg/${name}.svg` }];
    }
  );

  const { svgIconPacks } = useNebularIconPacks();
  svgIconPacks.set('ion', new Map(_ionicons));

  return (
    ...
  )
}
```

`ion` is the pack name, we can use ionicons like this:

```html
<Icon icon="logo-usd" pack="ion" />
```

## 3rd-party/Custom CSS Pack

In case you need to use any other 3rd party CSS Icon pack, here is the example of how to register font-awesome:

### Outside of NebularProvider

```ts
import { defaultFontIconPacks, NebularProvider } from '@nebular-react/styles';
import '@fortawesome/fontawesome-free/css/all.css';

export function App() {
  const fontIconPacks = new Map(defaultFontIconPacks);
  fontIconPacks.set('fa', { packClass: 'fa', iconClassPrefix: 'fa' });
  fontIconPacks.set('far', { packClass: 'far', iconClassPrefix: 'fa' });

  return <NebularProvider fontIconPacks={fontIconPacks}>...</NebularProvider>;
}
```

### In children components of NebularProvider

```ts
import { useNebularIconPacks } from '@nebular-react/styles';
import '@fortawesome/fontawesome-free/css/all.css';

export function SomeComponent() {
  const { fontIconPacks } = useNebularIconPacks();
  fontIconPacks.set('fa', { packClass: 'fa', iconClassPrefix: 'fa' });
  fontIconPacks.set('far', { packClass: 'far', iconClassPrefix: 'fa' });

  return (
    ...
  )
}
```

`fa` or `far` is the pack name, we can use ionicons like this:

```html
<Icon icon="anchor" pack="fa" /> <Icon icon="bookmark" pack="far" />
```
