# Create Nebular Page

Let's create a simple Nebular page (header + sidebar) in your project.
Add layout components with a sticky header into your `SomePage.tsx`:

```ts
import { Layout, Sidebar } from '@nebular-react/core';

export function SomePage() {
  return (
    <Layout
      header={<Layout.Header fixed>Company Name</Layout.Header>}
      sidebars={<Sidebar>Sidebar Content</Sidebar>}
      columns={<Layout.Column>Page Content</Layout.Column>}
    />
  );
}
```
